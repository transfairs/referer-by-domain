# AMO Summary

The "Summary" field on the addon's listing page (250-character limit). Already at the cap, so the 2.0.0 UI additions (theme, language override, modals, import/export) aren't called out here — see [description.md](description.md) for those.

## 🇬🇧 English

Referer By Domain gives you fine-grained control over how your browser sends referrer headers. You can disable referrers for specific domains, send only the origin, send the full URL, or leave referrers unrestricted. Improve privacy + compatibility.

## 🇩🇪 German

Referer nach Domain gibt Ihnen die volle Kontrolle darüber, wie Ihr Browser Referrer-Header sendet. Deaktivieren Sie Referrer für einzelne Domains, senden Sie nur den Ursprung oder die volle URL, oder lassen Sie Referrer uneingeschränkt zu.
